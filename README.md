# OpenClaw Desktop Icon

This project installs a Linux desktop launcher named `OpenClaw Chat`.

When you click it, it does three things:

1. Checks whether the OpenClaw gateway is already running.
2. Starts the gateway if it is not running.
3. Opens OpenClaw chat in your default browser using `openclaw dashboard`.

It is designed to be safe to click repeatedly and to reuse an existing running OpenClaw instance when available.
